# cs593-project
Project for CS593 at Purdue

## Requirements

### Equipment
1. Furhat robot, virtual or physical
2. Internet connection
3. Webcam stream

### Software
1. Python 3
2. Packages in `requirements.txt`
3. Ability to run `yolo-v8-nano` without problems at 15FPS


## Running the code
```
usage: gui.py [-h] [--camera CAMERA] [--fps FPS]

Webcam viewer

options:
  -h, --help       show this help message and exit
  --camera CAMERA  Camera index
  --fps FPS        Target FPS
```
